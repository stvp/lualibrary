--- title: hincrbyex brief: Increment a value in a hash only if it exists updated: 2014-07-31 10:28:00 layout: script sha: 3cb0d65c399fda8a1d4eb93b2fe85311671bc969 return: The new value of the counter if found, otherwise nil contributors: bpo --- Keys: 1
  1. A hash containing a field to be used as a counter.
Args: 2
  1. The field within the hash to use as a counter.
  2. The amount to increment the counter by.

Example

> evalsha 3cb0d65c399fda8a1d4eb93b2fe85311671bc969 1 guests sally 1
(nil)
> hset guests sally 8
(integer) 1
> evalsha 3cb0d65c399fda8a1d4eb93b2fe85311671bc969 1 guests sally 1
(integer) 9
> evalsha 3cb0d65c399fda8a1d4eb93b2fe85311671bc969 1 guests sally 1
(integer) 10

Source

if redis.call("HEXISTS", KEYS[1], ARGV[1]) == 1 then
  return redis.call("HINCRBY", KEYS[1], ARGV[1], ARGV[2])
else
  return nil
end
